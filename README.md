## Descrição da Solução

O ChargeGrid Intelligence é uma plataforma para gerenciamento inteligente de estações de carregamento de veículos elétricos.

A solução busca otimizar o uso da energia elétrica por meio de gerenciamento de demanda, tarifação dinâmica, interoperabilidade entre equipamentos e aplicação de inteligência artificial para auxiliar na tomada de decisão.

Esta Sprint apresenta uma prova de conceito funcional desenvolvida como protótipo digital.

---

## Objetivo do Projeto

Desenvolver uma solução capaz de:

* Gerenciar a demanda energética dos carregadores;
* Aplicar tarifação por horário (Time-of-Use);
* Permitir interoperabilidade entre diferentes sistemas;
* Utilizar inteligência artificial para recomendações e otimização de carga;
* Facilitar o acompanhamento das estações por meio de dashboard.

---

## Funcionalidades Implementadas

### Dashboard

* Monitoramento de carregadores ativos;
* Visualização da demanda atual;
* Receita diária estimada;
* Indicadores ambientais.

### Tarifação Inteligente

* Tarifas por faixa horária;
* Simulação de recarga;
* Estimativa de custo;
* Sugestão de economia baseada em IA.

### Gerenciamento de Demanda

* Controle da demanda contratada;
* Priorização de carregamento;
* Smart Charging.

### Interoperabilidade

Protocolos considerados:

* OCPP 2.0.1
* OCPI 2.2
* ISO 15118
* OpenADR
* REST API
* MQTT

### Métodos de Pagamento

* PIX
* Cartão NFC
* Aplicativo
* RFID
